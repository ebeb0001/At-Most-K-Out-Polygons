Welcome to At-Most-K-Out polygons

Given a finite set of points S, in the plane, a simple polygon of S is a non-self-intersecting polygon whose vertices are points from S. An at most k-out polygon is such a polygon in which at most k points of S lie outside the polygon. All other points are either vertices or lie inside its interior.

This concept conveniently connects different well-known classes of polygons:

  • k = 0: surrounding polygons of S (all points are inside or on the boundary)
  • k = n - 3: all simple polygonizations of S

What's available on this website ?

This site a small companion to a research project on enumerating at at-most-k-out polygons. It focuses on two main parts:

  • Playground - an interactive canvas where you can:
    • Draw point sets
    • Choose a value for k
    • Run the enumerator to see all at most k-out polygons it finds
    • Inspect statistics like the number of polygons, th number of inside/outside points and more
  • Algorithm - a more theoritical page that explains:
    • The definition of at most k-out polygons
    • The parent child structure between polygons
    • the operations used by the algorithm
    • How the implementation explores the "family tree" of polygons

There is also an About link with the demo and the reference paper we used to create this webpage.
